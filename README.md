# CompanyManagement
Company Management project solution
Доброго Дня!
Виконав тестове завдання.Назвав проект CompanyManagement.Виконав проект за доп. ASP.NET MVC на C#.
Особливості роботи програми:
На початку завантажується каталог,в якому кнопкою  "Add new root/child Company" створюємо основну компанію.
Щоб додати до якоїсь компанії дочірню,натискаємо на неї мишкою і кнопкою "Add new root/child Company" додаєм нову компаніюб
задаємо їй імя.
Щоб задати компанії Earnings, натискаємо на неї мишкою і далі кнопку "CompanyEarnings".У вікні що появилось,вводим суму.
Так само чиним коли треба редагувати суму виручки.
Щоб змінити назву компанії,двічі натискаємо на ній правою кнопкою миші і відкривається вікно редагування.
Щоб видалити компанію(і дочірні компанії),робим її активною і натискаємо відповідну кнопку.
Додатково я майже реалізував можливість drug and drop перетягування вузлів у дереві компаній.

Щоб його запустити,треба створити БД з назвою CompanyManagement і запустити sql запит в папці Models(в проекті).
Ну і ,можливо,підправити connectionString у WebConfig файлі.
